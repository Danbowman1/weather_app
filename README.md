# Basic Weather App

#### What I used:
* HTML
* Bootswatch
* JavaScript
* Weatherstack API

#### Apps Function
##### In this app you can pick the city you live in and it will return:
* City Name
* State Name
* Weather description
* Temperature
* An icon for the weather
* Relative Humidity 
* Precipitation
* What the temp. feels like
* Wind Speed
